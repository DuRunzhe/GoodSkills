# 五台山 Playbook

> 中国四大佛教名山之首,文殊菩萨道场。梁思成 1937 年在此发现佛光寺东大殿(中国现存最早唐代木构建筑)。

## 概要

**核心吸引力**:
- **古建筑**:佛光寺东大殿(唐) / 南禅寺(唐,中国最古木构) / 显通寺(青铜殿) / 塔院寺(大白塔)
- **佛教文化**:文殊菩萨道场 / 五爷庙(求财最灵) / 殊像寺(最大文殊铜像)
- **梁思成朝圣**:佛光寺被誉为「中国第一国宝」,建筑史必去

**适合**:2-3 天 / 自驾 / 历史文化爱好者 / 不想人挤的深度游

**不适合**:纯自然风光诉求 / 商业化诉求(台顶寺庙商业化较重) / 完全夜生活(山区夜生活少)

## 专属规则(LLM 必读)

- **素斋时段**:寿宁寺 / 普化寺素斋仅 11:30 和 17:30 两个时段,错过吃不上(必须排进 D1 午餐 / 晚餐)
- **必去点权重**:佛光寺 1.0(国保中的国保) / 南禅寺 0.8(顺路 5km) / 显通寺 0.7 / 塔院寺 0.6(地标) / 五爷庙 0.5(人多)
- **避免**:商业化严重的「五爷庙」人挤 / 南山寺白天去(夜晚才出彩)
- **疲劳管理**:D1 行程 6h 寺院游,需中间休息(下午茶 / 咖啡)
- **季节性约束**:海拔 1500-2500m,凌晨 10°C,需带冲锋衣(无论季节)
- **D2 起早**:为错峰 + 看古建最佳光,D2 应 05:30 起床,06:00 出发
- **返程硬约束**:13:30 必须返,15:00 后京昆高速易堵车,提前 2h 出保险

## 季节性

- **最佳月份**:6-9 月(避雨季 7-8 月底)
- **冬季**:11 月-3 月封山,部分寺院关闭(佛光寺可能因雪封路)
- **节庆**:
  - 农历四月初四:文殊诞辰(法会,人多)
  - 六月:五台山国际旅游月(人多)
  - 七月十五:盂兰盆节(法会)
