## 为什么去

五台山是中国四大佛教名山之首、文殊菩萨道场,藏传汉传佛教并存。海拔 1500-2500m 的清凉世界,夏季均温 20°C,是避暑+深度古建的稀缺组合。

但真正让五台山封神的不是佛教,而是**古建筑**:1937 年梁思成在此发现**佛光寺东大殿**,断代为唐构(857 年),被誉为「中国第一国宝」,改写了中国建筑史。亚洲现存最古木构是隔壁的**南禅寺**(782 年),17 尊唐代塑像真品,顺路 5km 必去。

台怀镇核心寺院群(显通寺 / 塔院寺 / 菩萨顶 / 殊像寺)虽商业化较重,但**显通寺的青铜建筑群**国内独一份,**塔院寺大白塔**是五台山视觉地标。2-3 天深度游是黄金比例,1 天太赶、4 天以上会审美疲劳。

## 路线总览

```
D1: 北京(顺义) → 京昆高速 → 保定服务区 → 忻州 → 五台山台怀镇
    → 五爷庙(广化寺) → 显通寺 → 塔院寺 → 寿宁寺素斋 → 五台山宾馆
D2: 五台山宾馆 05:30 起 → 佛光寺(07:30 开门) → 南禅寺 → 龙泉寺
    → 台怀镇蒙汉餐厅午餐 → 殊像寺 → 菩萨顶 → 普化寺素斋 → 南山寺夜景
    → 五台山宾馆
D3: 五台山宾馆 → 罗睺寺(可选) → 京昆高速服务区 → 石家庄午餐
    → 北京(顺义)
```

全程约 720km,环线不走回头路。

## 🎒 行前必带

### 证件
身份证、驾驶证、行驶证、车辆保险单(电子保单即可)

### 装备
- 冲锋衣 + 抓绒(无论季节,凌晨 10°C)
- 登山鞋 / 防滑运动鞋(寺院石阶多)
- 防晒霜 + 墨镜(高海拔紫外线强)
- 雨伞 / 雨衣(夏季雷阵雨)
- 保温杯(喝热水)

### 衣物
- 分层穿法:短袖 + 薄针织 + 冲锋衣
- 长裤(防蚊 + 草地露水)

### 药品
- 肠胃药(蒙餐油腻)
- 感冒药(温差大)
- 红景天 / 葡萄糖(高反预防,非必需)
- 创可贴 + 碘伏(徒步擦伤)
